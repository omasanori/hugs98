Hugs98 for .NET examples

The examples/ directory contains some simple demos of the .NET extensions:
  • Env.hs -- static method invocation example.
  • Http.hs -- slurping URLs.
  • Mail.hs -- sending mail using the .NET framework classes.
  • Delegate.hs -- wrapping up Haskell function value as a delegator; demonstrates in the context of an ultra-simple WinForms C# application.
  • CallIn.hs -- low-level example of how to call into Haskell from .NET code.
  • NewObj.hs -- shows how to dynamically generate a .NET class whose methods are all implemented by Haskell functions.
Note:For reasons yet unknown, you need to cd into the installation directory of Hugs98 for .NET before running the Delegate example: 
c:\> cd c:\Program Files\Hugs98.NET
c:\Program Files\Hugs98.NET> hugs examples/Delegate
